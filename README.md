# Banking APP Project

A simple banking application built with Python, demonstrating Object-Oriented Programming (OOP) concepts using Class and Subclass.

## Features

- User registration & login with password retry limit and auto-logout
- Account management (deposit, withdrawal, transfer)
- Transaction history query by time range
- Financial products with fixed-term interest calculation
- Blacklist/whitelist for secure transfers
